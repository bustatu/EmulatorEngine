# Emulator Engine

Use build.sh to build on Linux and build.bat to build on Windows.

## Currently supported platforms:
- Bytepusher
- CHIP8 
    - might have some unsuported SCHIP or XOCHIP opcodes, open an issue if so
- Gameboy (soon)
- NES (soon)
